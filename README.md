# 游戏说明

## 游戏名称
中文名：末世双雄
英文名：Two against the world
## 游戏概述
《双人打僵尸》是一款本地双人合作射击游戏，玩家需要共同对抗不断涌现的僵尸，生存下来并获得高分。游戏采用8方向移动系统，武器朝向随移动方向变化，支持自动射击和自动复活等特色功能。

## 游戏操作
### 玩家1
- **移动**：WASD键
- **射击**：空格键
- **切换武器**：Q键（上一个）/ E键（下一个）
- **自动射击**：数字键4（切换）

### 玩家2
- **移动**：方向键
- **射击**：回车键
- **切换武器**：小键盘1（上一个）/ 小键盘2（下一个）
- **自动射击**：数字键4（切换，与玩家1共享）

### 作弊功能
- **数字键0**：玩家2 AI托管
- **数字键1**：切换无敌模式（开启时玩家不会受到伤害，血条保持满状态）
- **数字键2**：切换怪物静止模式（开启时僵尸停止移动）
- **数字键3**：清屏（秒杀所有怪物）
- **数字键4**：切换自动射击（开启后无需手动按键，自动持续射击）
- **数字键5**：切换无限子弹（开启后弹药不会被消耗）
- **数字键7**：切换友伤机制（开启后玩家可能会伤害队友）
- **数字键8**：开关地图调试模式
- **数字键9**：开关攻击范围显示
- **加号键（+）**：加速游戏（每次增加1倍速度，最高8倍）
- **减号键（-）**：减速游戏（每次减少1倍速度，最低1倍）
- **ESC键**：暂停/恢复游戏

## 游戏特色
1. **8方向移动**：支持斜向移动，武器朝向随移动方向自动调整
2. **多种武器**：包含手枪、霰弹枪、冲锋枪、狙击枪、火焰喷射器等多种武器
3. **自动射击**：开启后无需手动按键，自动持续射击
4. **自动复活**：靠近死亡队友（50像素范围内）即可开始复活，远离则自动中断
5. **自动回血**：每秒自动恢复1点生命值
6. **游戏日志**：实时记录游戏中的各种事件和状态变化
7. **僵尸种类**：包含普通僵尸和BOSS僵尸，BOSS会吃掉小僵尸增强自身
8. **地形障碍**：游戏中会生成随机地形，影响移动和战斗策略
9. **游戏速度调节**：支持1-8倍速度调节，适应不同游戏节奏
10. **碰撞体积系统**：精确的碰撞检测，玩家不能穿过丧尸身体
11. **子弹纠偏**：子弹会根据最近敌人位置进行轻微纠偏，提高命中率
12. **油桶**：灰色圆柱体，一击就爆炸，对周围造成范围伤害
13. **急救包**：白色矩形带红色+号，拾取后恢复50点生命值
14. **换弹系统**：武器弹药耗尽时自动换弹，带有换弹声音和进度条
15. **攻击范围显示**：可显示僵尸的攻击范围，便于玩家躲避
16. **碰撞体积显示**：可显示玩家、僵尸和地形的碰撞体积
17. **友伤机制**：可通过数字键7开启/关闭，开启后玩家可能会伤害队友
18. **视觉描边**：玩家、僵尸、油桶和急救包都有黑色描边，增强视觉效果
19. **AI智能射击**：在友伤模式下，AI会避免射击到友方

## 游戏规则
1. **生命值**：每个玩家初始生命值为100，受到僵尸攻击会减少
2. **死亡与复活**：生命值为0时玩家死亡，可被队友复活
3. **游戏结束**：当所有玩家死亡时游戏结束
4. **分数系统**：消灭僵尸获得分数，分数显示在玩家信息面板
5. **无敌时间**：被攻击后会获得短暂的无敌时间，期间不会再次受到伤害
6. **油桶爆炸**：油桶被击中后会爆炸，对周围的敌人和玩家造成伤害
7. **急救包**：场地会小概率刷新急救包，拾取后恢复生命值

## 视觉效果
- **玩家颜色**：玩家1为红色，玩家2为蓝色，区分度高
- **血条显示**：位于玩家上方，显示当前生命值
- **伤害飘字**：受到伤害时会显示红色伤害数值
- **复活进度条**：复活队友时显示绿色进度条
- **自动射击指示器**：开启自动射击后玩家身上会显示红色小点
- **换弹进度条**：换弹时显示红色进度条
- **攻击范围**：显示僵尸的攻击范围（红色半透明圆圈）
- **碰撞体积**：显示玩家、僵尸和地形的碰撞体积（红色线条）
- **油桶**：灰色圆柱体，带有顶部和底部的圆形效果
- **急救包**：白色矩形，带有红色+号

## 游戏优化
1. **性能优化**：限制最大僵尸数量为15，每2帧更新一次UI，减少DOM操作
2. **游戏平衡性**：僵尸生成间隔会逐渐缩短，增加游戏难度
3. **用户体验**：阻止方向键和空格键的默认行为，避免页面滚动
4. **子弹纠偏**：提高玩家命中率，增强游戏体验

## 技术实现
- **前端技术**：HTML5 Canvas + JavaScript
- **游戏循环**：使用requestAnimationFrame实现流畅的游戏循环
- **碰撞检测**：基于矩形和圆形碰撞检测算法
- **事件处理**：使用键盘事件监听处理玩家输入
- **音效**：使用Web Audio API实现游戏音效
